# 岩波理恵デジタルファンクラブ

岩波理恵さん（昭和歌謡のプロ歌手）のデジタルファンクラブ PWA アプリです。

## 機能

### 公開ページ
- **ホーム** - アーティスト写真 + メニュー + 新着情報
- **最新ニュース** - 徳間ジャパンから自動取得
- **イベント情報** - ライブ・イベント告知
- **YouTube** - サムネイル付き動画一覧
- **個人ブログ** - Amebloから自動取得

### 会員限定
- **ファンクラブ会報** - デジタル会報バックナンバー
- **会員認証** - 会員番号 + パスワード

## 技術構成

- Next.js 14 (App Router)
- Tailwind CSS
- Vercel デプロイ
- サーバーサイドでのデータ取得（ISR: 1時間キャッシュ）
- PWA対応

## セットアップ

```bash
npm install
npm run dev
```
